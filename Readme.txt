This is how you can run this project:
Educational Purpose: Database and Management System

1. Download and Unzip the file on your local system copy bbdms.
2. Put bbdms folder inside root directory

Database Configuration
Open phpmyadmin "http://localhost/phpmyadmin"
Create Database bbdms
Import database bbdms.sql (available inside package)
Open Your browser put inside browser “http://localhost/Bloodbank-and-Donar-Management-System/bbdms/”

For Admin Panel
Open Your browser put inside browser “http://localhost/bbdms/admin”
